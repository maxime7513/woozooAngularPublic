Comment sa marche ?

  • notification

  • coordonnées

  • récupération

  • livraison

Je reçois une notification quand une course est disponible et dans ma zone de travail, à ce stade je peux accepter ou refuser la misson.

Je reçois les coordonnées (coordonnées clients, complément d'adresse) nécessaires à la récupération du colis jusqu'à sa livraison.

Je me rend à l'adresse de récupération du colis, pour valider cette étape :

  1. Je scanne le code barre de sécurité
  2. Je prend en photo le colis

Je me rend à l'adresse du destinataire, pour valider ma livraison :

  1. Je rentre le code de sécurité donné par le destinataire du colis
  2. Je scanne le code barre de sécurité
